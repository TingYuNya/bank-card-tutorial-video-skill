# 图片隐私检查契约

## 必须逐张检查

每个 `manifest.json` 资产都要在 `privacy-review.json` 中存在记录，且 `reviewed=true`。

## 默认遮挡

- 完整卡号、CVV、有效期、PIN。
- OTP、登录验证码、安全问题答案。
- Routing number、account number、IBAN。
- 姓名、详细地址、邮箱、手机号、生日。
- 身份证件、税号、客户编号。
- 二维码、条形码和可复用登录链接。
- 真实余额、信用额度、账单金额和交易记录。

## 坐标

```json
{
  "x": 0.1,
  "y": 0.2,
  "w": 0.4,
  "h": 0.08,
  "label": "card_number"
}
```

坐标以原图左上角为原点。`x+w` 与 `y+h` 不得超过 1。

## 遮挡方式

- `solid`：高风险数据默认使用，最终画面用不透明深色块。
- `blur`：只用于低风险背景信息。

## 复核

最终视频抽帧后再次检查。图片缩放、裁切和镜头移动可能导致遮挡偏移，源图审核通过不能替代最终帧复核。
