# Support

## 支持范围

本项目提供 Markdown 教程到视频成片的本地工作流，维护范围包括：

1. 项目初始化、数据契约和校验脚本。
2. ElevenLabs、OpenAI TTS 和 Azure Speech 的接口适配。
3. 字幕生成、分镜审核、时间线预览和视频渲染。
4. 隐私遮挡、关键帧抽检和质量报告。
5. Codex 与 Claude Code 的 Skill 使用说明。

以下内容需要使用者自行处理：

1. TTS、模型和云服务的账户、费用、配额及地区限制。
2. 银行账户申请、身份验证、信用审批和交易争议。
3. 具体银行产品信息的最终确认。
4. 第三方图片、字体、音乐、声音和商标的使用授权。
5. 操作系统、显卡驱动和复杂本地环境的通用维护。

## 提问前准备

提交 Issue 时请提供：

1. 操作系统和 Python 版本。
2. FFmpeg 与 ffprobe 版本。
3. 使用的 TTS 提供商。
4. 执行命令和完整错误信息。
5. 可公开的最小复现项目。
6. 已脱敏的配置和日志。

请删除 API Key、Token、银行卡号、CVV、验证码、账户号、真实地址和未脱敏账单。

## 问题分类

- 程序报错或行为异常：使用 Bug report。
- 新功能、接口或流程建议：使用 Feature request。
- 安全漏洞或敏感信息泄露：按照 `SECURITY.md` 私密报告。
- 银行产品事实修正：附上官方页面、协议或费率表，并标明核验日期和适用地区。

## 版本信息

Issue 标题建议包含版本号，例如：

```text
[v0.1.0] 时间线预览拖动后字幕延迟
```
