# Roadmap

本文件记录计划方向，不代表交付承诺。优先级会根据真实教程项目中的失败案例调整。

## 近期

- 增加可复现的无外部 API 演示项目和示例成片。
- 为分镜、事实核验和隐私审核 JSON 增加更严格的 Schema 校验。
- 增加字幕安全区、字体缺失和长文本溢出的自动检查。
- 增加 Windows、macOS 和 Linux 的环境诊断说明。
- 增加 GitHub Release 打包脚本和校验和文件。

## 中期

- 增加录屏片段与静态截图混合时间线。
- 增加可选的局部模糊、像素化和纯色遮挡策略。
- 增加多语言配音稿和字幕派生流程。
- 增加 TTS 提供商失败后的可恢复任务队列。
- 增加事实来源到具体口播句的可视化追踪。

## 暂不纳入

- 自动代替人工确认金融产品条款。
- 自动发布到视频平台。
- 自动上传真实银行卡、账单或身份文件到第三方服务。
- 复杂多机位剪辑、专业调色和完整 NLE 工程文件输出。
